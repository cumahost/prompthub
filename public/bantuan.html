Panduan Isi Prompt

Prompt Artikel

  • Topik: Tema utama artikel (misal: cara memulai bisnis online).
  • Target Audiens: Untuk siapa artikel ini ditujukan (UMKM, pelajar, dll).
  • Gaya Penulisan: Naratif, edukatif, profesional, santai, dll.
  • Tujuan Artikel: Edukasi, promosi, motivasi, dll.
  • Panjang Artikel: Opsional. Contoh: 800 kata.

Prompt Ilustrasi

  • Objek Utama: Apa yang ingin digambar (contoh: ibu UMKM membuka laptop).
  • Setting/Latar: Tempat atau suasana (rumah, toko, pasar).
  • Gaya Visual: Flat design, kartun, realistis, dll.
  • Warna Dominan: Opsional, misalnya biru, hijau, pastel.
  • Tujuan Gambar: Untuk banner, blog, sosial media, dll.

Prompt Kode / Script

  • Bahasa: PHP, Python, JavaScript, dll.
  • Fungsi: Apa yang ingin dicapai dari kode itu.
  • Fitur Khusus: Validasi, animasi, koneksi API, dsb.
  • Output: Apakah tampil di web, CLI, file?

Prompt Toko Online / Website

  • Jenis Bisnis: Apa produk atau layanan yang dijual?
  • Platform: CMS (WordPress + WooCommerce) atau custom HTML + PHP.
  • Fitur yang Diinginkan: Keranjang, pembayaran, testimoni, dll.
  • Target Pasar: Lokal, nasional, internasional, usia, gender, dll.

Prompt Surat / Dokumen

  • Jenis Surat: Undangan, pengantar, pernyataan, kontrak.
  • Ditujukan Kepada: Instansi, organisasi, individu.
  • Format & Bahasa: Formal, santai, bahasa baku, dll.
  • Isi Inti: Tujuan dari surat tersebut.
